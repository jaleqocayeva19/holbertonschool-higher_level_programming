# JavaScript Warm Up

This project is an introduction to JavaScript programming using Node.js.  
It focuses on basic concepts such as variables, loops, functions, objects, and modules.

---

## Learning Objectives

At the end of this project, you should be able to understand:

- How to run JavaScript with Node.js
- Variables (`const`, `let`)
- Data types
- Conditions (`if / else`)
- Loops (`for`, `while`)
- Functions
- Objects and arrays
- Command line arguments (`process.argv`)
- Module exports and imports

---

## Requirements

- All scripts are interpreted on Ubuntu 20.04 LTS using Node.js (v14)
- All files must end with a new line
- The first line of all files must be: `#!/usr/bin/node`
- Code must be semistandard compliant
- No use of `var`
- All files must be executable

---
